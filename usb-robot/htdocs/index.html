usb-robot

usb-robot is a program is to communicate with generic USB devices using libusb (which only works under linux at the moment). It can be used to send commands to the device directly from the console (usb-robot-slave), or as a backend to some rapidly prototyped driver in a language like perl or Python. At the moment it only supports control and bulk transfers.

It comes with a program (usb-robot-master) that translates traces of MS-Windows driver actions from the output of sniffusb version 0.1 into usb-robot-slave commands, so you can get your device to do whatever MS-Windows told it to.
